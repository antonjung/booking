# overview
a pwa app to manage room bookings at the village hall although this should be configurable to be any venue with multiple rooms or facilities

# detail
there soudl be a log in with multiple user types:
admin - can add new users
controller - can authorise bookings
booker - can view hall bookings and make booking requests

# requests
should be for a date, time and duration (in half hour slots) and a facility

when made the controller receives a notification or email with details
controller authorises or denies booking
on authorisation/denial the booker receives a notification or email with details

# users
user profile includes organisation, email address, name, phone number, contact preference (email/notification/both)

# facilities

usually a room or the whole hall (which requires all rooms to be available)
could be such things as bar staff, projector etc
